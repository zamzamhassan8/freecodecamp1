Introduction

JavaScript is a powerful scripting language used for web development.

It allows you to make interactive websites.

console.log("Hello, world!");
  • Easy to learn
  • Runs in browser
Basics

JavaScript syntax is similar to other C-like languages.

Statements end with semicolons.

let x = 10;
  • let
  • const
  • var
Functions

Functions are reusable blocks of code.

You define them with the function keyword.

function greet() { alert("Hi"); }
  • Reusable
  • Scalable
Objects

Objects hold key-value pairs.

They are used to store structured data.

let user = { name: "John", age: 30 };
  • Flexible
  • Nested
Arrays

Arrays are lists of values.

They can store any data type.

let nums = [1, 2, 3];
  • Indexed
  • Length property
